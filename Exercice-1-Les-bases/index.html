Les balises

Les balises paires

Elles s'ouvrent, contiennent du texte, et se ferment plus loin.

  • titre (h1, h2, h3, ...)
  • paragraphe (p)
  • section (div, span)
  • liste (ul,ol,li)
  • corps du document (body)

Les balises orphelines

Ce sont des balises qui servent le plus souvent à insérer un élément à un endroit précis (par exemple une image).
Il n'est pas nécessaire de délimiter le début et la fin de l'image, on veut juste dire à l'ordinateur « Insère une image ici ».

  • saut à la ligne (br)
  • image (img)
  • lien vers un fichier (link)

Les attributs

Les attributs sont un peu les options des balises. Ils viennent les compléter pour donner des informations supplémentaires.
L'attribut se place après le nom de la balise ouvrante.

La différence entre inline et block

block : une balise de type block sur votre page web crée automatiquement un retour à la ligne avant et après. Il suffit d'imaginer tout simplement un bloc.
Votre page web sera en fait constituée d'une série de blocs les uns à la suite des autres.
Mais vous verrez qu'en plus, il est possible de mettre un bloc à l'intérieur d'un autre, ce qui va augmenter considérablement nos possibilités pour créer le design de notre site ! Source

  • Exemples de balises block
    • paragraphe (p)
    • titre (h1, h2, h3, ...)
    • pied de page (footer)

inline : une balise de type inline se trouve obligatoirement à l'intérieur d'une balise block.
Une balise inline ne crée pas de retour à la ligne, le texte qui se trouve à l'intérieur s'écrit donc à la suite du texte précédent, sur la même ligne (c'est pour cela que l'on parle de balise « en ligne »). Source

  • Exemples de balises inline
    • image (img)
    • structuration du texte (abbr, q, ...)
    • tableau (table)

Le markdown et l'HTML

            
Markdown :
                Insérer une image : ![Texte alternatif](url_de_l'image "texte pour le titre, facultatif")
                Insérer une citation : > Ce texte apparaîtra dans un élément HTML <blockquote>
                Insérer une vidéo youtube : [texte du lien](url_du_lien "texte pour le titre, facultatif")
            

        
                
HTML :
                    Insérer une image : <img src="url_de_l'image ou lien relatif" alt="texte alternatif "/>
                    Insérer une citation : <blockquote> pour les citations "longues", <cite> pour les citations du titre d'une œuvre ou d'un évènement et <q> pour les citations "courtes"
                    Insérer une vidéo youtube : <object width="largeur de la vidéo" height="hauteur de la vidéo" data="url de la vidéo"></object>
